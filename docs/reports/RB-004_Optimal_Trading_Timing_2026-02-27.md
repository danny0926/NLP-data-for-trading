## 方法論說明

### 雙視角分析框架

| 視角 | 進場日 | 衡量目標 | 適用場景 |
|:-----|:-------|:---------|:---------|
| 議員視角 | Transaction Date | Informed trading alpha | 理解議員資訊優勢 |
| 跟單視角 | Filing Date | Follower alpha | 實際可操作的進場時機 |

### 計算公式

```
Excess Return = Stock Return - SPY Return
Directed Return = Excess Return × Direction (Buy=+1, Sale=-1)
CAR(t) = Σ Daily Directed Excess Return from T+0 to T+t
Sharpe = Mean(Directed Return) / Std(Directed Return)
Annualized Sharpe = Sharpe × √(252 / holding_days)
Profit Factor = Σ(Winning Trades) / |Σ(Losing Trades)|
```

### 資料限制

1. **資料期間短**: 2025/12 ~ 2026/02 (約 3 個月)
2. **Filing date 集中**: 88% 的 filing 在 2026-02-27，跟單視角分析不完整
3. **未計交易成本**: 佣金、滑價、借券費、稅
4. **存活偏差**: 僅分析有 ticker 的交易
5. **市場環境**: 分析期間為特定市場條件，結果可能不具普遍性
