# Research Briefs

研究團隊產出的 Research Brief 存放於此目錄。

## 命名規則

`RB-{序號}_{議題名稱}.md`，例如：
- `RB-001_SEC-Form4-Integration.md`
- `RB-002_PostgreSQL-Migration.md`

## 狀態

| 狀態 | 說明 |
|------|------|
| Draft | 研究中 |
| Reviewed | research-lead 已審閱 |
| Approved | C-Suite 已核准，準備 POC |
| Rejected | 不執行，附理由 |
| Completed | POC 已完成 |
